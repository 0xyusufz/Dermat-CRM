# Automated Demat Clinic

Premium SaaS CRM for dermatology clinic management.

## Tech Stack

- React + TypeScript
- Tailwind CSS v4
- Shadcn-style UI components
- Framer Motion
- Lucide Icons
- React Router

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Features

- Executive dashboard with KPIs, sparklines, and activity feed
- Patient registration with validation
- Consultation workflow with patient search
- All patients CRM with filters
- Premium patient profile with tabs and timeline
- Active & completed prescriptions
- Follow-up management (today, upcoming, missed, completed)
- Dark mode, global search, notifications
- Dummy data only — no backend required
